# Creating AWS Environment for Testing Purposes
## Tools Used

* **Terraform** (*IaC*)
* **AWS** (*Cloud Provider*)
### Optional
* **GitHub** (*SCM*)
* **VSCode** (*IDE*)

Some basic _**Terraform**_ commands are
```
terraform init 
terraform plan
terraform apply
terraform fmt
terraform destroy
terraform state list
terraform apply --auto-approve
terraform show <resource>.<resource_name>
```
